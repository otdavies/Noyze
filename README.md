# Noyze

A browser-based audio effects processor built with Rust (WASM) and React.

**[Launch App](https://otdavies.github.io/Noyze/)**

## Features

- Real-time audio processing powered by Rust/WASM
- Modifying effects: Saturation, Stereo Width, Transient Shaper, Sub Bass, Air, Tape Emulation
- Mastering effects: Multiband Compressor, Limiter, Soft Clipper
- Randomizer with curated "vibe" presets and blending
- Waveform visualization and transport controls
- Drag-and-drop audio file loading
